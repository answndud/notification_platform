# Appendix B - 명령어 치트시트

- 상태: 초안 완료
- 목표 분량: 2쪽

## Topic
- `kafka-topics --create`
- `kafka-topics --describe`
- `kafka-topics --list`

## Producer/Consumer
- `kafka-console-producer`
- `kafka-console-consumer --from-beginning`

## Group/Offset
- `kafka-consumer-groups --describe`
- `kafka-consumer-groups --reset-offsets`

## Config
- `kafka-configs --describe`
- `kafka-configs --alter`
