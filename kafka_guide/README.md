# Kafka 가이드북 작업 공간

이 디렉토리는 한국어 독자를 위한 교과서형 Kafka 가이드 집필 공간입니다.

## 목표

- Kafka를 처음 접하는 독자가 끝까지 따라올 수 있는 입문서 작성
- 총 분량 목표: 약 300쪽
- 형식: 챕터 기반 교과서(개념, 실습, 요약, 연습문제, 체크리스트)

## 사용 방법

1. `kafka_guide/WRITING_PLAN_TODO.md`에서 전체 진행 상태를 확인합니다.
2. 기본적으로 번호 순서대로 집필하되, 우선순위에 따라 조정합니다.
3. 집필 세션이 끝날 때마다 챕터 상태와 체크리스트를 갱신합니다.
4. 한국어 문체 가이드는 `kafka_guide/STYLE_KR.md`를 따릅니다.

## 파트 구성

- Part 02: 핵심 구성 요소
- Part 03: 명령/패턴
- Part 04: 아키텍처/운영
- Part 05: 성능/보안
- Part 06: 프로젝트 적용
