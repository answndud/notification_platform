# Appendix C - 연습문제 정답 가이드

- 상태: 초안 완료
- 목표 분량: 1쪽

## 사용 방법

- 정답은 "한 가지"가 아니라 "근거 있는 선택"이 핵심입니다.
- 아래 기준으로 스스로 답안을 점검하세요.

## 채점 기준(셀프 체크)

1. 개념 정확성
- 용어 정의가 맞는가

2. 설계 근거
- 왜 그 선택을 했는지 설명 가능한가

3. 운영 고려
- 장애/성능/보안 관점이 포함되었는가

4. 일관성
- 키 규칙, TTL, 정책이 충돌하지 않는가

## 모범 답안 작성 팁

- 1문항당 5~10줄로 간결하게 작성
- "상황 -> 선택 -> 이유 -> 위험 -> 보완" 순서 권장
